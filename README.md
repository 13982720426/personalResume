## 快速开始

1. `npm i` 安装依赖
2. `npm run dev` 开发调试
3. 编辑 `/src/index.html` 文件
4. `npm run build` 构建
5. 浏览器打开 `/index.html` 查看产物

> Chrome 浏览器 > 打印 > 更多设置无边距 > 另存为 PDF > 导出
